# 💬 Django Real-Time Chat Application

A web-based real-time chat application built using Django. Users can send and receive messages instantly with dynamic updates without refreshing the page.

---

## 🚀 Features

- User authentication (Login / Register)
- Real-time messaging experience
- AJAX-based asynchronous updates (no page reload)
- Clean and responsive UI
- Message storage using MySQL database
- Session-based user management

---

## 🛠️ Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, JavaScript (AJAX)
- **Database:** MySQL
- **Tools:** Git, VS Code
